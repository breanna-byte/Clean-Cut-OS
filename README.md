OPEN IT

2. Customer quote requests are saved in that browser.
3. Open Business Dashboard. Default PIN: 1027.
4. Go to Settings immediately and enter your business email, rates, and a new PIN.

IMPORTANT LIMITATIONS
- This version has no cloud database. Data stays in the browser/device where it was entered.
- A customer using the public site on their own device cannot automatically place the request into your dashboard. They can email/copy it; a true shared workflow requires a backend.
- Photo analysis is not AI yet. Photos are stored locally and must be reviewed by a person.
- The PIN is convenience protection only because all code runs in the browser. Do not store alarm codes, lockbox codes, IDs, banking data, medical information, or other sensitive information.
- Use Export Backup regularly. Clearing browser storage can erase local records.
